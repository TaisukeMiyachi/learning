## 紹介と使い方

  - 学生の自宅学習の効果・学力の可視化を行うゲーム。

  - 「学力」「眠気」という２つのパラメータを利用し、学習と休憩のバランスが大切であるという視点を入れる。
  - 休憩の必要性を「眠気が100以上になったらゲームオーバー」という設定を入れ実現した。
  -　一発逆転要素として「SNS」を入れ、ゲーム、ランダム要素入れた。

## 苦戦した点
